# VSCode — Configuration et extensions

## Mes extensions favorites

| Extension | Pourquoi |
|-----------|----------|
| ESLint | détecte les erreurs JS direct |
| Prettier | formate le code tout seul |
| GitLens | voir l'historique dans l'éditeur |
| EditorConfig | suit les règles du projet |
| Markdown Preview | aperçu en direct des .md |

## Raccourcis que j'utilise tout le temps

- `Ctrl+P` → ouvrir un fichier vite
- `Ctrl+Shift+P` → palette de commandes
- `Ctrl+\`` → terminal intégré
- `Alt+↑/↓` → déplacer une ligne
- `Ctrl+D` → sélectionner le mot suivant

## Mes réglages
