# Visual Studio Code

VS Code est un éditeur de code gratuit, puissant et très utilisé par les développeurs. Il offre une grande flexibilité grâce à son système d'extensions.

---

## Extensions installées

### ESLint

Analyse le code JavaScript pour détecter les erreurs et appliquer les bonnes pratiques. Indispensable pour écrire un code propre et cohérent.

### Prettier

Formate automatiquement le code (indentation, guillemets, points-virgules). Fonctionne avec la plupart des langages.

### GitLens

Ajoute des informations Git directement dans l'éditeur : historique, auteur des modifications, annotations. Très utile pour comprendre l'évolution d'un fichier.

### EditorConfig for VS Code

Applique automatiquement les règles définies dans le fichier `.editorconfig` du projet (indentation, encodage, etc.).

### Markdown Preview Enhanced

Permet de prévisualiser les fichiers Markdown en temps réel pendant l'édition.

## Raccourcis clavier utiles

| Raccourci | Action |
|-----------|--------|
| `Ctrl + P` | Ouvrir un fichier rapidement |
| `Ctrl + Shift + P` | Palette de commandes |
| `Ctrl + \`` | Ouvrir / fermer le terminal intégré |
| `Ctrl + S` | Enregistrer le fichier |
| `Ctrl + Z` | Annuler la dernière action |
| `Alt + ↑ / ↓` | Déplacer une ligne vers le haut ou le bas |
| `Ctrl + D` | Sélectionner le mot suivant identique |
| `Ctrl + /` | Commenter / décommenter une ligne |
